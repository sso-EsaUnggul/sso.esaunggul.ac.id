Kendala E-Learning !



Panduan Penanganan Kendala Akses Website E-Learning

Apabila mahasiswa atau dosen mengalami kesulitan dalam mengakses website E-Learning, seperti halaman tidak dapat dibuka, tampilan tidak muncul dengan sempurna, upload tugas tidak berhasil, atau proses login gagal, maka disarankan untuk melakukan langkah-langkah penyelesaian berikut:

  1. Gunakan Browser Alternatif
    Pastikan Anda mencoba membuka website menggunakan browser lain seperti:

    • Google Chrome

    • Mozilla Firefox

    • Microsoft Edge

    • Safari (untuk pengguna Mac dan iPhone)

Beberapa kendala akses dapat terjadi karena perbedaan kompatibilitas antara sistem dengan browser yang digunakan.

  1. Gunakan Mode Private/Incognito Tab
    Mode ini membantu menghapus sementara cache, cookies, serta data sesi lama yang dapat mengganggu proses login atau tampilan website.

Berikut cara mengaktifkannya pada masing-masing browser:

  • Google Chrome / Microsoft Edge: Tekan Ctrl + Shift + N (Windows) atau Command + Shift + N (Mac).

  • Mozilla Firefox: Tekan Ctrl + Shift + P (Windows) atau Command + Shift + P (Mac).

  • Safari: Pilih menu File → New Private Window pada Mac, atau buka Private Tab di tampilan tab pada perangkat iPhone/iPad.

  1. Catatan Tambahan:
    Jika setelah melakukan langkah-langkah di atas website masih tidak dapat diakses, pastikan koneksi internet stabil dan coba bersihkan cache browser secara manual melalui pengaturan browser yang digunakan.
